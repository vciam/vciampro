Core Application Flow

User Interface Layer

Consists of `index.html` (entrypoint) and all React components under `/components`.

  • [EVENT] User types in `textarea` or clicks a tag.
  • [EVENT] User clicks "Generate" button.
  • [EVENT] User imports a `.json` file.
  • [EVENT] User selects text in the output.

State & Logic (`App.tsx`)

The monolithic root component that acts as the central hub for state management and business logic.

  • [WORKFLOW] Updates React state (`content`, `selectedTags`).
  • [WORKFLOW] Triggers `handleSubmit()` function.
  • [WORKFLOW] Triggers `handleImportKeywords()` function.
  • [WORKFLOW] Renders `SuggestionDisplay` with new props.

Service Layer (`geminiService.ts`)

Encapsulates all communication with the external Gemini API. Responsible for prompt engineering.

  • [SSOT] `getOrganizationSuggestions` assembles a prompt using `content`, `tags`, and `customTags`.
  • [SSOT] `discoverKeywordsFromText` analyzes imported text.
  • [SSOT] `analyzeAndSuggestContextualImprovements` performs meta-analysis on user context.

Google Gemini API

The external AI model that processes the prompts and returns the evolved content.

  • Receives the structured prompt from the Service Layer.
  • Processes the request using either `gemini-2.5-flash` or `gemini-2.5-pro`.
  • Returns a response containing the evolved text, keywords, and semantic score.
